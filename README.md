# projeto_django

## Introdução

**Nome do projeto:** blog_django  

**Contexto:** Desenvolvido como um projeto pessoal e de aprendizado em Django, com foco em criação de um blog funcional de tecnologia da informação.  

**Detalhes relevantes:** Este é um projeto inicial em Django, que serve como base para desenvolvimento de aplicações web completas. Foi motivado pelo interesse em consolidar conhecimento em Python, Django, HTML, CSS e integração de front-end e back-end.

---

## Principais Funcionalidades do Projeto

- **Blog de Posts:** Exibição de posts de tecnologia da informação em cards estilizados, com títulos, subtítulos e resumos.
- **Navegação:** Menu funcional com links para páginas Início, Sobre e Contato.
- **Página Sobre:** Conteúdo informativo sobre o blog, com imagens e listas de tópicos de TI.
- **Página Contato:** Formulário simples para envio de mensagens (simulado com HTML neste projeto).
- **Layout Responsivo:** Utilização de CSS para garantir boa visualização em desktop e mobile.
- **Estrutura Django:** Uso de `manage.py`, banco de dados SQLite (`db.sqlite3`) e app `blog`, permitindo futura expansão do sistema.

**Interface:** O usuário visualiza posts em cards, com sidebar de categorias e posts populares, podendo navegar entre páginas do blog de maneira intuitiva. A página inicial contém banner, posts recentes e área de newsletter.

---

## Tecnologias Utilizadas

- **Python:** Linguagem principal para o backend, utilizada para desenvolver a lógica do Django.
- **Django:** Framework web usado para criar a aplicação, gerenciar rotas, views, templates e banco de dados.
- **SQLite:** Banco de dados local padrão do Django, utilizado para armazenar informações do blog.
- **HTML/CSS:** Estrutura e estilo das páginas web, incluindo layout responsivo, cards, menus e formulários.
- **Git/GitHub:** Controle de versão e hospedagem do código fonte.

## Conclusão

O **projeto_django** permitiu colocar em prática conhecimentos fundamentais de Django e desenvolvimento web.
